Welcome to Android 2 Arduino

This is an Android-Processing-Arduino application which makes a connection between an Android device and an Arduino using a server. The Android application creates a client socket in order to connect with the server and send some data which are taken from the accelerometer sensor of the device. The server is written in Processing language and creates a server socket which accepts all client socket calls and forwards the incoming messages to the Serial port. The Arduino application reads the values from the Serial port and does all some fancy things! (This application does not require to have any bluetooth or wifi accessory!)

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Android Application

The Android application connects to a server (in a computer) and sends some values. It created in order to make a connection between an Android device and an Arduino using a server. This application creates a socket and makes the connection with the server and sends some data which are taken from the accelerometer sensor of the device. If the user tilts (or shakes) the phone to the left the application sends a '1' to the server and if the user tilts the phone to the right the application sends a '0' to the server.

Processing Application

The Processing application is the "MeddleMan" to connect an Android device with the Arduino through the WiFi connection. It uses a Server Socket in order to let the Android device to communicate with it forwards the incoming message to the Serial port of the computer which is connected to the Arduino.

Arduino Application

The Arduino application takes some values from the Serial port and do different things depending on the values.

Troubleshooting

The Arduino and Processing applications are connected to each other with Serial port. In order to the Processing application be connected to the correct port you have to change the port list array to the one that you connect the Arduino in your computer.

Processing App

int mySerialPortList = 0;
String portName = Serial.list()[mySerialPortList];
